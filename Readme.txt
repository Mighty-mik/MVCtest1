
1.Pour que ça marche vous devez créer une base des données
 
2. Aller dans "app/", et ouvre le ficher "config.php"

3. Modifier les informations de conexion à la base de données.

4. Dès l'application est lancée, elle créer automatiquement une table sql "user"

5. Le fichier index.php comprend certainnes constantes

6. Le Router est composé de deux fichiers : le .htaccess et l'index.php
